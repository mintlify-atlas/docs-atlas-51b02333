Salvo is a powerful, ergonomic async web framework for Rust. Built on [Hyper](https://hyper.rs) and [Tokio](https://tokio.rs), Salvo makes it easy to write high-performance web services without sacrificing simplicity. Whether you need a simple REST API or a full-featured web application with WebSockets, TLS, and OpenAPI documentation, Salvo has you covered.

## Why Salvo?

Salvo takes a fresh approach to Rust web frameworks by unifying handlers and middleware into a single concept. You write the same kind of function whether you're handling a request or adding cross-cutting behavior. This eliminates the complexity of distinct middleware traits and makes code easier to compose and test.

## Key features

- **Handler-based middleware** — Middleware is just a handler. No extra traits or generics required.
- **Tree routing** — Apply middleware at any level of your route tree, not just globally.
- **First-class OpenAPI** — Add `#[endpoint]` to any handler and get OpenAPI docs for free.
- **HTTP/1, HTTP/2, HTTP/3** — Full protocol support out of the box with feature flags.
- **Auto TLS via ACME** — Automatically obtain and renew certificates from Let's Encrypt.
- **WebSocket & WebTransport** — Real-time protocols built into the framework.
- **Zero unsafe code** — `#![forbid(unsafe_code)]` enforced across the entire codebase.
