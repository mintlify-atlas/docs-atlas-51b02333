---
title: "Get started with Salvo in four steps"
description: "Create a new Rust project, add Salvo and Tokio dependencies, write your first handler with #[handler], and start an async HTTP server using Cargo."
---

Salvo gets you from zero to a running web server in a few lines of Rust. This guide walks through creating a new project, adding dependencies, writing a handler, and starting the server. By the end you'll have a working HTTP service ready to extend.

<Steps>
  <Step title="Create a new Rust project">
    Use Cargo to scaffold a new binary project:

    ```bash
    cargo new hello-salvo
    cd hello-salvo
    ```
  </Step>
  <Step title="Add dependencies">
    Add Salvo and Tokio to `Cargo.toml`. Enable the `oapi` feature if you want OpenAPI support later:

    ```toml Cargo.toml
    [dependencies]
    salvo = { version = "0.92", features = ["logging"] }
    tokio = { version = "1", features = ["macros", "rt-multi-thread"] }
    tracing-subscriber = "0.3"
    ```
  </Step>
  <Step title="Write your first handler">
    Replace `src/main.rs` with the following:

    ```rust src/main.rs
    use salvo::prelude::*;

    // A handler is just an async function decorated with #[handler].
    // Salvo injects only the parameters you ask for.
    #[handler]
    async fn hello() -> &'static str {
        "Hello World"
    }

    #[handler]
    async fn hello_zh() -> Result<&'static str, ()> {
        Ok("你好，世界！")
    }

    #[tokio::main]
    async fn main() {
        tracing_subscriber::fmt().init();

        let router = Router::new()
            .get(hello)
            .push(Router::with_path("你好").get(hello_zh));

        println!("{router:?}");

        let acceptor = TcpListener::new("0.0.0.0:8698").bind().await;
        Server::new(acceptor).serve(router).await;
    }
    ```
  </Step>
  <Step title="Run and test">
    Start the server:

    ```bash
    cargo run
    ```

    In another terminal, send a request:

    ```bash
    curl http://localhost:8698
    # Hello World
    ```
  </Step>
</Steps>

<Note>
  `use salvo::prelude::*` imports the most commonly used types — `Router`, `Server`, `TcpListener`, `#[handler]`, `Request`, `Response`, `Depot`, and more. You can import them individually if you prefer explicit imports.
</Note>
